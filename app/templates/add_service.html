API Documentation
Notification Endpoint:
POST /api/notify
Required Headers:
X-API-KEY: your_generated_api_key
Content-Type: application/json
Request Body:
{
  "message": "Your notification message here"
}
Example Usage:
curl -X POST http://your-domain/api/notify \ -H "X-API-KEY: your_generated_api_key" \ -H "Content-Type: application/json" \ -d '{"message": "System is running normally"}'
