Manual Chat Addition

If the automatic chat refresh doesn't work or you want to add chats manually, you can add them here. You'll need the chat ID from Telegram.

How to get a chat ID:
  1. Add your bot to the chat (group/channel) or start a private conversation
  2. Send a message in the chat
  3. Use the Telegram Bot API to get updates: https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
  4. Look for the chat.id field in the response
Add Chat Manually Click the button above to manually add chats.
Troubleshooting
Common Issues:
  • No chats appear: Make sure your bot is added to groups/channels
  • Bot kicked from chat: Remove the chat from the database and re-add it
  • Permission errors: Check if the bot has permission to send messages
Bot Requirements:
  • Bot must be added to the chat
  • Bot must have permission to send messages
  • For channels, bot must be an admin
  • Bot token must be valid and not expired
